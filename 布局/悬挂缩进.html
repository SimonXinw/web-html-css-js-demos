Hanging indent

参考文献式悬挂缩进

第二行起与首行左缘对齐的「编号 / 作者」突出：用左内边距配合负的 text-indent。

[1] 陈某某．悬挂缩进在数字出版中的应用研究[J]．出版科学，2023，31(4)：12–19．

[2] Bringhurst, R. The Elements of Typographic Style. 4th ed. Hartley & Marks, 2013.

[3] W3C. CSS Text Module Level 3. 建议用于理解 text-indent 与排版边界的关系。

p {
  padding-left: 2.25em;
  text-indent: -2.25em;
}
Hanging indent

Word 式悬挂缩进

首行向左「外凸」，其余行整体右移：左外边距 + 负首行缩进（与上一种视觉对调，注意使用场景）。

注：本段模拟合同或说明书中的条款编号样式。首行顶在更靠左的位置，后续行与正文左边界对齐，阅读时编号更醒目。

第二条　乙方应在约定交付日之前完成自检，并向甲方提交书面报告。若因不可抗力导致延期，须在事件发生之日起三日内通知甲方。

p {
  margin-left: 2.25em;
  text-indent: -2.25em;
}
Layout

栅格：编号列 + 正文列

需要严格对齐多位编号时，用 grid 比单段负缩进更稳。

12.

栅格左侧放序号，右侧放正文，天然悬挂且不会与长编号抢行。

12.3

子条款同样对齐；若换成 display: table 也可达到类似效果。

.row {
  display: grid;
  grid-template-columns: 3.25em 1fr;
  column-gap: 0.5em;
}
Paragraph

中文段落首行缩进两格

正文段落常用 text-indent: 2em，与悬挂缩进不同，但常一起出现。

传统中文排版中，段落首行空两格是惯例。现代网页可用 em 单位随字号缩放。

第二段继续缩进。注意标题、列表项通常不再做首行两格。

p { text-indent: 2em; }
Initial letter

首字下沉

浮动首字母 + 支持浏览器的 initial-letter 渐进增强。

版式中的首字下沉能引导视线进入长文。实现时注意行高与字重，避免与下一行粘连。西文段落效果更明显，中文亦可作为章节开场。

p::first-letter {
  float: left;
  font-size: 3.25rem;
  line-height: 0.85;
  padding-right: 0.18em;
}
@supports (initial-letter: 3) {
  p::first-letter {
    float: none;
    initial-letter: 3 2;
  }
}
Alignment

两端对齐

中文常用 text-justify: inter-ideograph；英文可开 hyphens（需 lang）。

在同一行宽下，两端对齐能让块面更整齐。中文排版中常配合适当行距与字号，避免字距被拉得过散。极窄容器下可改回左对齐以保可读性。

English sample

Justified text with hyphenation can improve the rag on the right edge when the measure is wide enough for the engine to break words cleanly.

.zh {
  text-align: justify;
  text-justify: inter-ideograph;
}
.en {
  hyphens: auto;
}
Columns

分栏排版

杂志式短栏；小屏自动降为单栏。

CSS Multi-column 适合展示型短文、术语表或侧栏注释。栏间距与分隔线能减轻拥挤感。

长文正文仍建议单栏控制行宽（measure），分栏更适合辅助阅读区块。

.cols {
  column-count: 2;
  column-gap: 2rem;
  column-rule: 1px solid #ddd;
}
Quote

侧栏引用

强调引文层级，与正文形成节奏对比。

Good typography reveals the meaning of the text; great typography makes the meaning unforgettable. — 演示文案，非原句出处 
blockquote {
  border-left: 4px solid var(--accent);

}
Definition list

定义列表 + 悬挂说明行

术语下挂解释：dd 使用与参考文献相同的缩进配方。

悬挂缩进（Hanging indent）
非首行相对首行增加左缩进，或首行相对正文「外凸」；具体实现依版式语义选择 padding / margin + text-indent。
Measure
单行字数与行长。中文常建议每行 25–35 字左右（视字号与载体而定），英文常以 45–75 字符为参考。
dd {
  padding-left: 1.25em;
  text-indent: -1.25em;
}
Verse

诗歌 / 行对齐

使用 white-space: pre-wrap 保留换行与空格。

轻叩纸张的边， 行距里藏着呼吸。 悬挂的一行， 托住整段沉默。

.verse {
  white-space: pre-wrap;
  line-height: 2;
}
List

自定义列表标记（悬挂）

取消默认 list-style，用伪元素对齐自定义符号。

  • 检查父级是否设置了 overflow: hidden，可能裁切负缩进段落。
  • 长编号优先用 grid 或 table 布局，而非硬编码 em 宽度。
  • 屏幕阅读器场景下保持语义列表结构，勿仅用 div 模拟。
li {
  position: relative;
  padding-left: 1.35em;
}
li::before {
  content: "—";
  position: absolute;
  left: 0;
}
Cheatsheet

排版属性速查

与悬挂、段落相关的常用属性一览。

属性 作用
text-indent 首行缩进；取负值配合 padding / margin 做悬挂。
padding-left 整块左内边距；与负 text-indent 组合成「首行突出」。
margin-left 整块右移；与负 text-indent 组合成 Word 式悬挂。
text-align: justify 两端对齐；中文常配 text-justify: inter-ideograph。
hanging-punctuation 标点悬挂（支持度有限，可作渐进增强）。
initial-letter 首字下沉（需 @supports 检测）。
字体：Noto Serif SC / Newsreader · Normalize：jsDelivr · 单文件演示，可直接复制各区块样式。
