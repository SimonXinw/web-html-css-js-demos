三、数据清洗 缺失值 / 重复值 / 异常值 / 类型转换 / 字符串
缺失值处理
检测 df.isna().sum()   # 每列缺失数量
填充 / 删除 df.fillna(0)   # 填充为 0
df.dropna()   # 删除含缺失的行
df["amount"].fillna(df["amount"].mean())   # 用均值填充
重复值处理
去重 df.drop_duplicates()   # 全行重复
df.drop_duplicates(subset=["order_id"])   # 按 order_id 去重
异常值处理
  • 箱线图：识别离群点
  • 3σ 原则：超出均值±3倍标准差视为异常
  • 业务规则：金额<0、日期不合理等直接过滤
过滤异常金额（0~10000） df = df[(df["amount"] >= 0) & (df["amount"] <= 10000)]
类型转换与字符串处理
类型转换 df["amount"] = df["amount"].astype(float)
df["order_date"] = pd.to_datetime(df["order_date"])
字符串（商品名转小写、去空格） df["product"] = df["product"].str.lower().str.strip()

四、数据聚合 groupby / agg / pivot_table / 交叉表
groupby 基础 核心
按用户统计订单数、总金额 df.groupby("user_id").agg(
  order_count=("order_id", "count"),
  total_amount=("amount", "sum")
)
多列聚合
多聚合函数 df.groupby("category").agg(
  {"amount": ["sum", "mean", "count"]}
)
pivot_table 透视表
类似 Excel 透视表，按行/列维度汇总。
按品类×月份汇总 GMV df.pivot_table(
  values="amount", index="category",
  columns=df["order_date"].dt.month, aggfunc="sum"
)
crosstab 交叉表
用户×状态交叉计数 pd.crosstab(df["user_id"], df["status"])

五、时间序列 日期解析 / resample / rolling / 同比环比
日期解析与提取
解析日期 df["order_date"] = pd.to_datetime(df["order_date"])
提取年/月/日 df["year"] = df["order_date"].dt.year
df["month"] = df["order_date"].dt.month
df["weekday"] = df["order_date"].dt.dayofweek
resample 重采样
按日汇总 GMV df.set_index("order_date").resample("D")["amount"].sum()
按月汇总 df.set_index("order_date").resample("M")["amount"].sum()
rolling 滚动窗口
7 日移动平均 df.set_index("order_date")["amount"].rolling(7).mean()
同比与环比
指标含义
环比与上一期（如上周、上月）对比
同比与去年同期对比
环比（shift 取上一期） df["amount_pct_change"] = df["amount"].pct_change()

八、电商数据分析实战 GMV / 转化率 / RFM
GMV
商品交易总额
df[df["status"]=="paid"]["amount"].sum()
已支付订单金额总和
转化率
CVR
下单用户数 / 访客数
需结合访问日志计算
AOV
平均客单价
gmv / 订单数
总金额 / 已支付订单数
RFM
用户价值分层
R=最近购买 F=频次 M=金额
Recency, Frequency, Monetary
GMV 与 AOV 计算
核心代码 paid = df[df["status"] == "paid"]
gmv = paid["amount"].sum()
order_count = len(paid)
aov = gmv / order_count
RFM 模型实现
计算 R / F / M rfm = df.groupby("user_id").agg(
  R=("order_date", lambda x: (today - x.max()).days),
  F=("order_id", "count"),
  M=("amount", "sum")
)
R/F/M 分箱 → 打标签 → 用户分层
品类销售分析
各品类 GMV 排名 df[df["status"]=="paid"].groupby("category")["amount"].sum()
  .sort_values(ascending=False)
