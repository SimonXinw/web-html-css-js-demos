GMV / Revenue 流量转化漏斗 用户 LTV / CAC 广告 ROAS / ACOS 物流履约 商品运营 财务利润链路 多平台对比

一、核心营收指标 衡量业务规模与盈利能力
GMV
商品交易总额 (Gross Merchandise Volume)
GMV = 订单数 × 客单价
含取消/退款的总交易额，衡量平台规模的首要指标
Revenue
实际收入 (Net Revenue)
Revenue = GMV - 退款 - 折扣 - 平台佣金
真正到手的钱，比 GMV 更能反映实际经营状况
净利润
Net Profit
净利润 = Revenue - COGS - 运营成本 - 税费
扣除所有成本后的最终盈利，企业存活的生命线
毛利率
Gross Margin
毛利率 = (Revenue - COGS) / Revenue × 100%
跨境电商通常 40%~70%，低于 30% 需警惕
AOV
平均客单价 (Average Order Value)
AOV = 总销售额 / 总订单数
提升 AOV 是提升 GMV 最高效的杠杆之一
ARPU
用户平均收入 (Avg Revenue Per User)
ARPU = 总收入 / 活跃用户数
衡量单个用户的变现效率，SaaS/订阅制尤为重要
GMV vs Revenue 的关系 必背
GMV（总交易额） → - 退款/取消 → - 折扣/优惠 → - 平台佣金 → Revenue（净收入）
面试常问 GMV 高不等于赚钱 → 如果退货率 30%，GMV 100 万实际只有 70 万有效
务必区分 GMV 口径：含税/不含税、含运费/不含运费

二、流量指标 用户从哪来、看了什么
UV
独立访客数 (Unique Visitors)
按设备ID / Cookie 去重
同一用户多次访问只计 1 次，衡量真实触达规模
PV
页面浏览量 (Page Views)
每次页面加载计 1 次
PV/UV = 人均浏览页数，反映用户浏览深度
Bounce
跳出率 (Bounce Rate)
跳出率 = 只看1页就离开的会话 / 总会话
电商首页 < 40% 为健康，> 60% 需优化落地页
Duration
平均停留时长 (Avg Session Duration)
总停留时间 / 会话数
电商 2~4 分钟为正常，内容页应更长
Session
会话数 (Sessions)
30分钟无操作则新开 Session
一个 UV 可能贡献多个 Session
Pages
每次访问页数 (Pages/Session)
PV / Sessions
越高说明商品浏览兴趣越强
流量来源分类 核心
来源说明典型 CPC
Organic SearchGoogle/Bing 自然排名（依靠 SEO）$0（免费）
Paid SearchGoogle Ads 关键词竞价（SEM）$0.5~$3
Social OrganicFacebook/TikTok/Instagram 自然流量$0（免费）
Social PaidFacebook/TikTok 付费广告$0.3~$1.5
Email邮件营销（EDM），复购利器$0.01~$0.05
Direct直接输入网址/书签，多为老用户$0
Referral联盟营销/外链跳转按 CPS 结算
UTM 参数追踪体系
URL 后附加 UTM 参数，让 GA4 等工具识别每一次访问的流量来源。
UTM 示例 https://store.com/shoes
?utm_source=facebook
&utm_medium=paid_social
&utm_campaign=summer_sale_2025
&utm_content=video_ad_v3
  • utm_source：来源平台（facebook / google / tiktok）
  • utm_medium：媒介类型（cpc / email / organic）
  • utm_campaign：活动名称，用于 A/B 对比
  • utm_content：素材变体标识，区分同一活动不同创意

四、用户价值指标 一个用户值多少钱
LTV
用户生命周期价值 (Lifetime Value)
LTV = AOV × 购买频次 × 留存时长
一个用户在整个生命周期内贡献的总利润
CAC
获客成本 (Customer Acquisition Cost)
CAC = 总营销费用 / 新增付费用户数
LTV/CAC > 3 为健康，< 1 则亏损获客
复购率
Repurchase Rate
复购率 = 有2次+购买的用户 / 总购买用户
品牌站 20%~40% 为优秀，铺货站通常 < 10%
留存率
Retention Rate
N日留存 = 第N日活跃用户 / 新增用户
次日/7日/30日留存，衡量用户粘性
NPS
净推荐值 (Net Promoter Score)
NPS = 推荐者% - 贬损者%
范围 -100~100，> 50 为优秀品牌体验
RFM
用户价值分层模型
R=最近购买 F=购买频次 M=消费金额
经典分层：高价值/潜力/流失预警/沉睡用户
LTV/CAC 黄金比率 必背
LTV/CAC 比值状态策略建议
< 1严重亏损获客立即止血，优化广告/提升客单价
1~3刚好打平或微利提升复购率、减少营销浪费
3~5健康可持续可适当加大投放力度
> 5投放不足增加获客预算，扩大规模
LTV 精细计算 LTV = Σ(每月 ARPU × 留存率n) / (1 + 折现率)n
简化版：LTV ≈ AOV × 月购买频次 × 平均留存月数 × 毛利率
用户留存曲线解读
将同一批次注册用户按时间追踪其回访/购买行为（Cohort Analysis）。
某品牌站月留存示例 第1月：100%（购买）
第2月：35% → 第3月：22% → 第6月：15% → 第12月：10%

曲线趋于平缓 → 有忠实用户基础 ✓
曲线持续下滑 → 产品/体验无法留住用户 ✗
  • Day 1 留存：产品首次体验是否吸引人
  • Day 7 留存：用户是否形成了使用习惯
  • Day 30 留存：是否有长期价值，可否投入复购营销

五、广告投放指标 钱花在哪、回报多少
CTR
点击率 (Click-Through Rate)
CTR = 点击数 / 曝光数 × 100%
衡量广告素材吸引力，越高越好
CPC
单次点击成本 (Cost Per Click)
CPC = 广告花费 / 点击次数
Facebook 平均 $0.5~$1.5，Google 搜索 $1~$5
CPM
千次曝光成本 (Cost Per Mille)
CPM = 广告花费 / 曝光数 × 1000
衡量品牌曝光效率，视频广告 $5~$15
CPA
单次转化成本 (Cost Per Action)
CPA = 广告花费 / 转化次数
Action 可以是购买、注册、加购等任意目标
ROAS
广告花费回报 (Return on Ad Spend)
ROAS = 广告收入 / 广告花费
ROAS 3x 意味着花 $1 赚 $3，盈利门槛通常 2x~3x
ROI
投资回报率 (Return on Investment)
ROI = (收入-成本) / 成本 × 100%
比 ROAS 多考虑产品成本，更接近真实盈亏
ACOS
广告销售成本比 (Ad Cost of Sales)
ACOS = 广告花费 / 广告收入 × 100%
Amazon 特有指标，ACOS = 1/ROAS，越低越好
TACOS
总广告销售成本比
TACOS = 广告花费 / 总销售额 × 100%
含自然订单的综合广告效率，Amazon 卖家必看
ROAS vs ROI vs ACOS 对比 必背
指标公式是否含成本使用场景
ROAS广告收入 / 广告花费仅广告费Facebook / Google Ads 通用
ROI(利润-投入) / 投入含全部成本整体投资回报分析
ACOS广告花费 / 广告收入仅广告费Amazon PPC 专用
TACOS广告花费 / 总销售额仅广告费Amazon 综合广告效率
换算关系 ACOS = 1 / ROAS × 100%
例：ROAS = 4x → ACOS = 25%
ROAS 看"赚了多少"，ACOS 看"花了多少"，本质等价
广告投放优化链路
提升 CTR → 降低 CPC → 提升 CVR → 降低 CPA → 提升 ROAS
  • CTR 低 → 素材/文案不吸引人，优化创意和定向
  • CPC 高 → 竞争激烈或受众过窄，拓展受众或换渠道
  • CVR 低 → 落地页体验差或价格无竞争力，优化商详页
  • CPA 高 → 整体投放效率低，需同时优化上下游

六、商品运营指标 管好货、卖好货
动销率
Sell-through Rate
动销率 = 有销量SKU数 / 总上架SKU数
> 70% 为健康，低则说明大量滞销商品
ITO
库存周转天数 (Inventory Turnover Days)
ITO = 平均库存 / 日均销量
越小越好，快消品 30~60 天，慢消品 90~180 天
退货率
Return Rate
退货率 = 退货订单数 / 总订单数 × 100%
服装 15%~30%，电子产品 5%~10%
BSR
畅销排名 (Best Sellers Rank)
Amazon 按品类实时更新
BSR 越小越畅销，是选品和竞对分析的核心参考
缺货率
Out-of-Stock Rate
缺货天数 / 总天数 × 100%
缺货直接损失销售额，Amazon 还会降低排名权重
滞销率
Dead Stock Rate
90天+无销量SKU / 总SKU
滞销库存占用仓储费，FBA 长期仓储费高昂
库存健康度评估 业务核心
库存状态判定标准应对措施
健康可售天数 30~60 天正常补货节奏
偏多可售天数 60~120 天促销清仓 / 减少补货
滞销可售天数 > 120 天大促清仓 / 移除 / 捆绑销售
缺货预警可售天数 < 14 天紧急补货 / 空运补仓
选品数据分析维度
  • 市场容量：品类月搜索量、BSR 排名区间、头部卖家数量
  • 竞争度：Review 数量分布、价格区间、新品上架频率
  • 利润空间：售价 - 采购成本 - 头程物流 - FBA费 - 平台佣金 - 广告费
  • 季节性：Google Trends 趋势、历史 BSR 波动
  • 差异化空间：现有产品差评痛点、功能改进机会

八、财务利润指标 赚不赚钱看这里
跨境电商利润链路 必背
售价 $49.99消费者支付金额
- 平台佣金 $7.50Amazon 约 15%
- FBA 费 $5.80配送 + 仓储

- 头程物流 $3.50海运分摊
- 广告费 $8.00PPC 投放
= 净利润 $13.19净利率 26.4%
盈亏平衡分析 (Break-Even)
盈亏平衡公式 盈亏平衡销量 = 固定成本 / (单价 - 单位变动成本)

例：月固定成本 $5,000，售价 $50，变动成本 $30
→ 盈亏平衡 = $5,000 / ($50 - $30) = 250 单/月
  • 固定成本：SaaS 工具费、人工、仓租、月度推广预算
  • 变动成本：采购、物流、平台佣金、包装、广告（按单分摊）
  • 贡献利润：单价 - 变动成本，用于覆盖固定成本
关键决策线 ROAS 盈利线 = 1 / 毛利率
例：毛利率 40% → ROAS > 2.5x 才盈利
ACOS 盈利线 = 毛利率 × 100%
例：毛利率 40% → ACOS < 40% 才盈利

十、指标关联与拆解 GMV = UV × CVR × AOV
GMV 核心拆解公式 必背
一级拆解 GMV = UV × CVR × AOV
二级拆解 — UV UV = 自然流量 UV + 付费流量 UV + 直接流量 UV
付费 UV = 广告曝光 × CTR
二级拆解 — CVR CVR = 商详浏览率 × 加购率 × 结账率 × 支付成功率
二级拆解 — AOV AOV = 件单价 × 每单件数
受品类结构、促销策略、捆绑销售影响
指标拆解树 — 诊断 GMV 下降
当 GMV 环比下降 15% 时，按拆解树逐层排查：
  • Step 1：GMV = UV × CVR × AOV → 哪个因子下降？
  • Step 2（UV 降）：拆分渠道 → 自然流量降？广告 CTR 降？预算削减？
  • Step 3（CVR 降）：拆分漏斗 → 加购率降？支付率降？页面加载慢？
  • Step 4（AOV 降）：拆分商品 → 低价品占比上升？优惠券力度过大？
  • Step 5：交叉验证 → 是否全品类/全渠道下降？还是某一品类/某一渠道？
实战提示 永远先看拆解，再看归因。不要上来就猜"可能是竞争对手降价了"
数据说话：UV 没变 + CVR 降了 → 问题出在站内体验，不是流量
利润拆解公式
利润完整链路 净利润 = GMV × (1 - 退货率) × 毛利率 - 广告费 - 固定成本

= UV × CVR × AOV × (1 - 退货率) × 毛利率 - 广告费 - 固定成本
  • 提升利润的 5 个杠杆：提升 UV、提升 CVR、提升 AOV、降低退货率、降低广告费
  • 每个杠杆提升 10% → 利润可能提升 50%+（乘数效应）
指标关联全景图
广告曝光 ×CTR→ UV ×CVR→ 订单数 ×AOV→ GMV → Revenue → 净利润
广告花费 ÷ Revenue → ACOS    Revenue ÷ 广告花费 → ROAS
核心公式速记 GMV = UV × CVR × AOV
Revenue = GMV × (1 - 退货率) - 折扣 - 佣金
净利润 = Revenue - COGS - 物流 - 广告 - 运营成本
ROAS = Revenue(广告) / Ad Spend   |   ACOS = 1/ROAS
LTV = AOV × 复购频次 × 留存周期 × 毛利率
健康标准：LTV/CAC > 3 & ACOS < 毛利率
